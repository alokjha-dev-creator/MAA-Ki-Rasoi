# 🍳 MAA Ki रसोई (Maa Ki Rasoi)

Welcome to **MAA Ki रसोई**, a vibrant, fully responsive multi-cuisine restaurant landing page designed to bring the warmth and comfort of a mother's kitchen right to your screen. 

This project features a curated showcase of global cuisines, including **Indian**, **Italian**, **South Indian**, and **Chinese** dishes, complete with direct navigation paths to detailed menus.

## 📱 Project Highlights & Layout Fixes

This repository serves as a major milestone in mastering **Mobile-First Responsive Web Design**. Key frontend challenges tackled include:
* **The Mobile Aspect Fix:** Resolved layout stretching and overflow issues by replacing rigid fixed-pixel values with fluid percentage-based constraints (`width: 100%`, `max-width: 100%`).
* **Adaptive Navigation:** Engineered a clean, sticky navigation bar that smartly stacks vertically on small mobile phone screens (like the Samsung Galaxy S8) for effortless thumb tapping, and gracefully expands horizontally across desktop browsers using CSS Flexbox.
* **Semantic HTML Structure:** Structured elements clearly using semantic boxes (`<nav>`, `<div>` wrapper cards) to keep code maintainable and search-engine friendly.

---

## 🛠️ Tech Stack Used

* **HTML5:** For semantic web layout, links, and structure.
* **CSS3:** For modern styling, color schemes (`bisque`, `beige`), custom box shadows, and fluid element sizing.
* **CSS Flexbox & Media Queries:** For cross-device layout adaptations and component alignment.
